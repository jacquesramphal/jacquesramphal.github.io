# Colors

All colors are defined as CSS variables in `src/app/globals.css`.
Edit those variables to retheme the entire system instantly.

## How to customize

Open `src/app/globals.css` and change the HSL values inside `:root`. All components pick up the change automatically.

```css
:root {
  /* Change this to your brand blue */
  --primary: 221 83% 53%;
  --primary-foreground: 0 0% 100%;

  /* Adjust corner radius across all components */
  --radius: 0.375rem;
}
```

HSL format is `hue saturation% lightness%` — omit the `%` signs in the CSS variable value itself (Tailwind adds them).
