# Welcome to your Design Environment

This is a blank Storybook starter configured with **Tailwind CSS** and **shadcn/ui** primitives.

## Getting started

1. **Customize tokens** — edit `src/app/globals.css` to change colours, radius, and spacing to match your brand
2. **Add components** — run `/design add [ComponentName]` to scaffold a new story
3. **Sync from composable-pro** — run `/design sync` when the team ships updated components

## What's included

| Path | What it is |
|------|-----------|
| `src/app/globals.css` | Design tokens (CSS variables) — start here |
| `src/components/ui/` | Component source files — copy and customize |
| `src/lib/utils.ts` | `cn()` helper for merging Tailwind classes |

## Token quick reference

Edit `src/app/globals.css` to change:
- `--primary` — main brand colour
- `--secondary` — secondary/neutral tone
- `--radius` — corner radius (applies to all components)
- `--background` / `--foreground` — page background and text
