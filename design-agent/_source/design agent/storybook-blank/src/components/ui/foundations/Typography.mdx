# Typography

The type scale is built on Tailwind's default font-size ramp with semantic utility classes applied via the `Typography` component (`src/components/ui/typography.tsx`).

There is no custom font configured by default — the system inherits the browser's system font stack. To add a custom font, load it in `src/app/globals.css` or via `next/font` and set `font-family` on `body`.

## Type scale

| Style | Class | Size | Weight | Use case |
|-------|-------|------|--------|----------|
| H1 | `TypographyH1` | `text-4xl / lg:text-5xl` | 800 | Page titles |
| H2 | `TypographyH2` | `text-3xl` | 600 | Section headings |
| H3 | `TypographyH3` | `text-2xl` | 600 | Subsection headings |
| H4 | `TypographyH4` | `text-xl` | 600 | Card or panel titles |
| Lead | `TypographyLead` | `text-xl` | 400 | Introductory sentences |
| P | `TypographyP` | `text-base` | 400 | Body copy |
| Large | `TypographyLarge` | `text-lg` | 600 | Callouts, pull quotes |
| Small | `TypographySmall` | `text-sm` | 500 | Labels, captions |
| Muted | `TypographyMuted` | `text-sm` | 400 | Metadata, timestamps |

## Adding a custom font

To use a Google Font or self-hosted font, update `src/app/globals.css`:

```css
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700;800&display=swap');

@layer base {
  body {
    font-family: 'Inter', sans-serif;
  }
}
```

Or use Next.js font optimisation in a layout file with `next/font/google`.
